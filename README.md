# Healthcare-Analytics
A Power BI dashboard that analyzes patient demographics, medical conditions, hospital billing, treatment costs, and admission trends across multiple years.


Project Overview

This dashboard provides a comprehensive view of healthcare operations, covering patient demographics, common medical conditions, hospital performance, admission trends, and billing analysis.
It helps hospitals and healthcare administrators understand:
* Who their patients are
* What medical conditions are most common
* Which hospitals handle the most patients
* How much revenue each hospital generates
* Seasonal or yearly admission trends

This type of dashboard supports resource allocation, budget planning, treatment optimization, and healthcare decision-making.
Tools Used
* Power BI – Data modeling, DAX, dashboards
* Excel – Data cleaning, preprocessing

How This Helps Healthcare Teams

* Identifies patient demographics & health trends
* Helps allocate doctors & hospital resources
* Tracks yearly admission patterns
* Highlights high-revenue hospitals
* Improves planning for common medical conditions

  

Supports decision-making for hospital management
